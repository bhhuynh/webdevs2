Chicago Crime Data Search

Hi this is a crime data search app for Chicago. The scroll bar allows you to select the crime you are looking for. We have created a link to the different districts so you can select your number. Remember to put a 0 before the number. Ex. (012, 008). Each district number should be 3 digits. If there is a Select one and click search. When searching for a new crime, refresh the page before doing so. Also we added a filter to limit the amount of results you can have. Input any number and it will display at most those number of results. Our second action is a reference to google maps based on the longitude and latitude values.
